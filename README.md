# Assets for interview

Fake data samples for Data Engineering Case Study interview.

## Data sources:
1. csv files:
    * `marketing_campaigns.csv` with campaign dates, budgets, and channels
    * `user_discounts.csv` with selected user_id discount data
2. proto schema files:
    * `shared.proto` with a shared event type definition for all schema envelopes
    * `article.proto` with the articles specification
    * `transction.proto` with the schema for transactions
    * `user.proto` with the specification for customers of the platform
3. json schema of frontend events
